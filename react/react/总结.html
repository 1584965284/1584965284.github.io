setState一定是异步吗？其实分成两种情况：（React18之前）

在组件生命周期或React合成事件中，setState是异步；

在Promise的回调、setTimeout或者原生dom事件中，setState是同步；

（React18之后）默认所有的操作都被放到了批处理中（异步处理）
